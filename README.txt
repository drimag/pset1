steps to run:
- install a C++ compiler and ensure that it has been added to PATH (verify with g++ --version)
- configure config.txt with 2 integers in this format: "<integer1> <integer2>"
    - each solution will have its own respective config file
- open the terminal in the folder of the solution you would like to compile and enter the following command:
    g++ -o primes.exe main.cpp util.cpp -pthread

- run the produced ".exe" file
